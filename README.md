## Install the Required Packages:
--------
### For Client
1. Open Terminal:

```bash 
cd client
```

`npm create vite@latest .`
Then Choose React, and JavaScript

`npm i axios chart.js zustand tailwindcss socket.io-client react-router-dom react-hot-toast react-dom react-chartjs-2 lucide-react framer-motion date-fns @hello-pangea/dnd`

```cd server```

`npm i express nodemon dotenv cors bycrypt helmet jsonwebtoken mongoose morgan multer jsonwebtoken node-cron socket.io zod`

2. In server Folder rename `.env.example` to `.env` and fill the detailes
